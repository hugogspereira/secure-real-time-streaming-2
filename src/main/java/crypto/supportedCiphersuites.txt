The supported ciphersuites for symmetric encryption:

AES/CTR/NoPadding-128
AES/CTR/NoPadding-256
AES/CCM/NoPadding-128
AES/CCM/NoPadding-256
RC6/CCM/NoPadding-128
RC6/CCM/NoPadding-256
RC6/CTR/NoPadding-128
RC6/CTR/NoPadding-256

In the handshake phase we can use the following digital signatures:

SHA256withRSA/PSS-2048
SHA256withRSA-2048
RSA-2048

It can be used:
DH-2048
NULL -> key generation through randoms generated by the server and box
